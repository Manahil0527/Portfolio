# Manahil Habib Portfolio

A premium, high-tech portfolio website built for an AI Engineer & Full Stack Developer.

## Features

- **Interactive Neural Network Background**: Dynamic particles connecting based on proximity, creating a "live grid" effect.
- **Custom Cursor**: A refined, two-part cursor system with smooth following and interaction states.
- **Background Textures**: Subtle grain and scan-line effects for a professional "sci-fi" aesthetic.
- **Interactive Ripple**: Visual feedback on every click.
- **SEO Optimized**: Semantic HTML and meta tags for better search engine visibility.
- **Responsive Design**: Fluid layouts that work on all screen sizes.

## Technlogy Stack

- **HTML5**: Semantic structure.
- **Vanilla CSS**: Premium gradients, animations, and glassmorphism.
- **Vanilla JavaScript**: High-performance canvas animations and DOM interactions.

## How to View

Simply open `index.html` in any modern web browser.
